# Typing Master

![Typing Master](screenshot.png)  
※スクリーンショットは任意

## 概要
Typing Masterは、ブラウザ上で遊べるタイピング練習サイトです。  
- 日本語文章や単語をローマ字入力で練習  
- 長文モードと単語モードに対応  
- 入力ミス・正確率・タイピング時間をリアルタイムで計測  
- タイピング終了後に結果画面を表示  

学習者やタイピング速度向上を目指す方に最適です。

---

## 特徴

- **クラス管理による問題管理**
  - `problems.js` で長文・単語問題を管理
  - 問題の追加・削除が簡単
- **ランダム出題**
  - 長文モード・単語モードごとにランダムで問題を選択
- **リアルタイム判定**
  - 入力した文字が正しいか即時判定
  - 日本語文字を2文字換算で正確率計算
- **結果画面**
  - タイピング終了後、ミス数・正確率・タイピング時間を表示
- **UI/UX**
  - TailwindCSSを使用したスタイリッシュなデザイン
  - 長文や単語練習に対応
  - 途中でやめるボタン（オプション）も実装可能

---

## ファイル構成

typing-site/
├─ index.html ← メイン画面（HTML + Tailwind）
├─ problems.js ← タイピング問題クラスと問題リスト
└─ script.js ← タイピングロジック（判定・結果画面）


---

## 使い方

1. `index.html` をブラウザで開く  
2. ホーム画面で「長文モード」または「単語モード」を選択  
3. 表示された日本語文章をローマ字で入力  
4. 入力終了後、結果画面でミス数・正確率・タイピング時間を確認  
5. 「ホームに戻る」で再度練習可能  

---
